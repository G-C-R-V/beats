Panel administrativo

Centraliza la carga de beats, previews de audio y paquetes de licencia. Todo se guarda localmente para que puedas probar la experiencia del cliente final.

Registrar un nuevo beat

Completa los datos, sube el arte, la preview de audio y asigna las licencias disponibles para este lanzamiento.

Licencias disponibles

Activa las licencias que incluye este beat. Cada licencia debe tener un precio, archivos descriptivos y un paquete de entrega.

La información, imágenes y archivos se almacenan en el navegador para propósitos demostrativos.
